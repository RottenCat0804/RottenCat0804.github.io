## 細かい構成とか

### Github

作ったプログラムを公開できるサイト。プログラマーの聖地。

今回のサイトはGithubでデータを管理してあります。誰でも見れます。

あとブラウザで作業を完結できるのも地味に便利。（本当はやっちゃいけないけど…）

### Github Pages

上記のGithubのおまけ機能で、サイトを公開できる。

制約はあるものの、ブログ程度のものであれば難なく動く。そして無料。

今回はサイトの公開のために使用。（クレカがないのでサーバーを借りれない…）

ページを一枚一枚作らなくても、本文だけ書いたらいい感じに整形してくれるイイ！やつ。

Github Pages がデフォルトで対応してくれている。
